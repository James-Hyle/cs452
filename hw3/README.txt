USAGE:
To use this program, cd into the folder "hw3" and invoke "gcc - o ldeq.o -c deq.c" as well as the "make" command at the terminal. The program will compile and then invoke "./Test/run shell" on the terminal to run the test suite. To check the program with valgrind invoke "./vg". To clean dir, invoke "make clean" on command line. Note that input redirection is not supported. To test '&' on shell command invoke "sleep 5 &" in shell CLI and you will notice that the shell prompt will come back right away.
